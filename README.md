The project is configured for automated deployment to GitHub Pages. Follow these steps to get your site live:

### 1. Local Development
To run the project locally for further development:

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

### 2. Build & Deploy
Once your code is on GitHub, use the following commands to build and publish your site:

```bash
# This command builds the project and pushes the 'dist' folder to the 'gh-pages' branch
npm run deploy
```

### 3. GitHub Configuration
After running `npm run deploy`, GitHub will automatically create a `gh-pages` branch.
1. Go to your repository on GitHub.
2. Navigate to **Settings > Pages**.
3. Ensure the **Source** is set to "Deploy from a branch".
4. Select the **`gh-pages`** branch and the **`/(root)`** folder.
5. Save changes. Your site will be live at `https://murdpra.github.io/portfolio/`.
